# Meetie

밋티는 IT 직군을 위한 스터디 플랫폼으로,
신뢰를 기반으로 한 학습 환경과 지속적인 인사이트 공유를 돕습니다.
함께 학습하며 더 단단해지는 커뮤니티를 경험해 보세요!

> 💡 내 관심사에 맞는 스터디를 찾기가 어렵지 않나요? \
> 🔒 익명 운영은 신뢰를 낮추고, 스터디의 질을 떨어뜨리기도 하죠. \
> 🌐 결과 중심의 스터디는 장기적인 네트워킹과 인사이트 공유가 부족합니다.

이런 고민들을 해결하기 위해 밋티가 만들어졌습니다.

IT 직군에 특화된 스터디를 통해 공통 관심사로 연결됩니다. \
회원 정보를 활용해 신뢰 지표를 제공하여 더 안전한 스터디 환경을 제공합니다. \
커뮤니티 네트워킹으로 지속적인 정보 교류와 학습을 촉진합니다. \
밋티와 함께라면, IT 직군 학습자들이 신뢰를 쌓고 서로의 성장을 돕는 스터디 문화를 만들어갈 수 있습니다.

테스트 계정 정보: test123@test.com / test123!
